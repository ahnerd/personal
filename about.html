Voila la liste de quelques services que je peux vous offrir: Développement et hébérgement d'un site web statique --------------------------------------------------- Un site web statique est un ensemble de fichiers écrits en languages HTML,CSS et JavaScript.Il n'a y as pas du PHP,Python ou n'importe quel langage du coté serveur c'est pour cela un site statique ne peut étre convenable que pour certains cas d'utilisation ,par exemple si vous avez besoin d'un site web pour assurer votre identité en ligne comme un site vitrine de présentation de votre entreprise aux visiteurs ou encore un site de portfolio ou vous pouvez présenter vos travaux aux clients potentiels. Un site web statique nécessite une intervention manuelle(changement de fichiers sources) chaque fois que vous voulez faire un changement dans votre site car il ne posséde pas une interface d'administration (back office) c'est pour cela il n'est pas convenable pour n'importe quelle entreprise. mais il'est moins couteux que les autres types de sites web que ce soit pour l'hébergement ou encore les frais du développeur web. Si vous n'avez besoin que d'un site web statique (selon vos besoins) alors je vous propose mes services pour la création de votre site qui inclut: L'achat et la configuration du nom de domaine de votre choix . La création des pages et l'hebergement de site web statique . Attention: Un site web statique n'est pas toujours un choix convenable pour chaque entreprise.Si vous avez besoin de plus d'une identité en ligne ou une présentation de votre entreprise,vos services et vos information de contacte vous devez considérer un site web dynamique ou encore une application web. Développement d'un site web dynamique ou une application web sur mesure avec Django ou Nodejs ------------------------------------------------------------------------------- Contrairement à un site web statique, un site dynamique est dévelopé avec des technologies du coté client(HTML,CSS et JavaScript) et des technolgies du coté serveur.Les technolgies serveur que je maitrise et je vous propose sont: Le langage Python en utilisant le framework Django. JavaScript avec Node.js et express.js Avec un site dynamique vous aurez une interface d'administration (back office) à votre disposition pour controller plusiers aspects plus le contenu de votre site quand vous voulez sans avoir besoin d'un savoir ou connaissance technique et sans savoir besoin de manuellement changer les fichiers sources de votre site chaque fois que vous envisagez un changement. Si vous avez de plus d'un site dynamique et si vous chercher une gestion informatique de votre entreprise donc vous devez opter pour une application web qui vous offre une présentation en ligne de votre entreprise plus une gestion locale ou à distance de votre entreprise et plusieurs autres avantages.
